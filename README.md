# GRAVEYARD SHIFT 🕯

a zine for the sleep-deprived and superstitious — a campy, GeoCities-era horror blog.

Static site, no build step. `index.html` is the whole show.
Authoring happens locally via a keeper's admin panel (not deployed).
`posts.js` (optional, not present by default) can bake posts into the public site via `window.GS_POSTS`.

best experienced with the lights off.
